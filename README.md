# Sales Data Analysis Dashboard

## Project Overview

This Power BI dashboard provides end-to-end sales analytics and business performance insights across products, regions, and customer segments.

## Business Objectives

* Analyze sales trends
* Track profitability
* Identify top-performing products
* Monitor regional performance
* Compare sales growth over time

## Tools Used

* Power BI
* DAX
* SQL
* Power Query

## Dashboard Features

* Sales Trend Analysis
* Profit vs Sales Comparison
* Regional Performance Analysis
* Product Category Insights
* Interactive Filters & Slicers

## Key KPIs

* Total Sales
* Total Profit
* Net Sales
* Profit Margin
* Top Products
* Sales by Region

## Insights

* Certain product categories generated high sales but low profitability.
* Regional sales trends showed seasonal fluctuations.
* Top-performing products contributed significantly to overall revenue.
